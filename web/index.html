Gchat

A beautiful, fast, privacy-conscious client for the Google Talk instant messaging service


Tweet

Metro

  • Modern design that integrates with the rest of Windows Phone
  • Uses your personal accent color and native smilies and icons

Tiles & Notifications

  • Recieve IMs even if the app isn't running
  • Pin your friends to start and recieve tile notifications for each one

Privacy & Security

  • Completely encrypted conversations using SSL and AES
  • Supports Google Talk's "off the record" conversations
  • Authentication done through Google: we never see your password

Free & Open Source

  • Gchat is and always will be open source
  • Get the code for the client and the server
